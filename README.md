# P2P文件传输工具

基于frp xtcp功能实现的P2P文件传输工具，支持大文件传输、断点续传、传输速度显示、双向传输等功能。

## 功能特点

- 支持大文件传输
- 断点续传功能
- 实时显示传输速度
- 支持双方互传文件
- 接收方确认后才开始传输
- 美观的用户界面（马卡龙暖色系）
- 易于操作的图形界面

## 技术特点

- 基于frp的xtcp功能实现P2P连接
- 使用Python语言开发
- PyQt5实现图形界面
- 支持打包成单个exe文件

## 使用方法

1. 下载并安装程序
2. 运行程序，选择角色（发送端/接收端）
3. 输入连接参数（密钥和服务名称）
4. 等待连接建立
5. 开始传输文件

## 注意事项

- 发送端和接收端必须选择不同的角色
- 双方必须使用相同的密钥和服务名称
- 确保网络环境稳定
- 建议在局域网环境下使用

## 开发环境

- Python 3.8+
- PyQt5 5.15.6+
- Windows操作系统
